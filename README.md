# MarketFusion

**MarketFusion** — сервис объединённой аналитики и управления ассортиментом для маркетплейсов **Wildberries** и **Ozon**.

Единая точка сбора данных по товарам, заказам, остаткам и продажам. Предоставляет чистый REST API и готовую основу для дашбордов, автоматизаций и аналитических систем.

## Особенности

- Унификация данных из Wildberries и Ozon в единую модель  
- Полная OpenAPI-документация (Swagger UI)  
- Чистые DTO, валидация, глобальная обработка ошибок  
- Готовый слой для JWT-авторизации (включается одной строкой)  
- Расширяемая архитектура под новые маркетплейсы  
- Docker Compose для быстрого старта  

## API и документация

Swagger UI сразу после старта:
➜ http://localhost:8080/swagger-ui/index.html  
OpenAPI spec:
